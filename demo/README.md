# Demo assets

`demo.tape` renders the README GIF deterministically with
[VHS](https://github.com/charmbracelet/vhs) — no manual screen recording.

```bash
# from the repo root
vhs demo/demo.tape          # writes demo/vibe-coding-demo.gif
```

Commit the regenerated `vibe-coding-demo.gif` alongside any change to `demo.tape`.
If you don't have VHS, record the same commands manually at ~20pt font and trim to
under 20 seconds.
